# git入门

## 扩展项
```
# 只更新工作区和暂存区，不产生新的提交。
-n，--no-commit
# 在提交信息的末尾追加一行操作者的签名，表示是谁进行了这个操作。
-s，--signoff
# 忽略路径中的转义字符
git config --global core.protectNTFS false
# 禁用换行符转换
git config --global core.autocrlf false
```

## Git 创建仓库
```git
**参数说明：**
* repo:Git 仓库。
* directory:本地目录。
#repo:Git 仓库。
directory:本地目录。
# 当前目录
git init
# 指定目录
git init newrepo
# 从现有仓库拷贝
git clone <repo>
# 拷贝到指定目录
git clone <repo> <directory>
# 添加远程仓库
git remote add <shortname> <url>
# 删除远程仓库
git remote rm name
# 修改仓库名
git remote rename <name> <new_name>
# 修改仓库地址
git remote set-url <name> <newurl>
# 添加仓库推送地址
git remote set-url --add <name> <newurl>
```

## 查看状态
```git
# -s 简短的结果
# --cached 尚已缓存的改动
# HEAD 查看已缓存的与未缓存的所有改动
# --stat 显示摘要
git diff 
```

## 添加到快照
```git
# path为.添加所有文件到缓存（可把.换为具体文件名，多个文件名用空格分割）
git add  <path>
# 回滚添加操作
git reset
```

## 添加到仓库
```git
# -a 自动添加快照再添加到仓库
# -m [备注]
git commit 
# 回滚最近一次提交
git reset --soft HEAD^
# 修改commit
git commit --amend
# 不触发检测提交
git commit --no-verify
```

## 代码回滚
```
# 放弃本地修改回退到指定分支
git reset --hard <ORIG_HEAD>
# 保持本地修改
git reset --merge <ORIG_HEAD>
```

## 分支管理
```
# 默认查看本地检出分支
# -a，all查看所有分支
# -m 修改分支名 <oldName> <newName>
# -r 查看远程分支
git branch

# 关联本地分支到线上分支
git branch --set-upstream-to=origin/dev dev

# 切换分支
git switch <branchName>
```

## 代码提交
```
# -f,force 强制覆盖线上分支
# -d,delete 删除分支
git push <远程主机名> <本地分支名>:<远程分支名>
```

## 代码合并
```
# 合并所有代码变动
git merge <barchName>
# 查看记录 可以看到commitHarsh
git log 
# 合并指定提交
git cherry-pick <commitHash>
```

# ssh
```
# 远程路径构成
<userName>@<host>:<url>

# 登录
# -p,port 指定端口
ssh <userName>@<host>

# 跨机远程拷贝 secure copy
# -r 	递归复制整个目录。  拷贝文件夹需要该参数
# -P port 	指定数据传输用到的端口号
scp [可选参数] file_source file_target 
```
