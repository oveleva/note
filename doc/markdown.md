# 「markdown」
## 标题
```markdown
标题：# 
注：多少个#代表多少级标题（1～6）
```

## 列表
```markdown
无序列表：- 或 * 
有序列表：1. 2. 3.
```

## 复选框
```markdown
复选框：- [] 
复选框选中：- [x]
```

## 引用
```markdown
> 这是引用的文字
注：多少个 > 代表多少级引用
```

## 图片与链接
```markdown
# text 描述 link 链接
链接：[text](link)
图片：![text](link)
注：需要控制图片大小请使用 <img src="" alt="test" style="zoom:50%" />
```

## 强调
```markdown
斜体：*斜体*
粗体：**粗体**
```

## LaTex 公式
```markdown
行内公式：
  $ y = x + 1 $
公式块：
  $$ a^2 + b^2 = c^2 $$
```

## 代码
`需要引用时可以用` \``将语句包起来。`
`如果引用代码，可以将`\`\`\``置于这段代码的首行和末行。`
```javascript
行内代码:
  `var a=1` 
代码区块：
  `` `ruby
  print 'Hello world'
  `` `
```
